# proposal feedback

Overall Feedback

These are two interesting environments that can be very challenging for algorithms, making a good choice for the project.

However, using just 2 algorithms for the benchmark is not enough. I would recommend using, at the very least, 3 algorithms. If time permits, experimenting with a fourth algorithm would enrich the project further, but comparing 3 algorithms and doing a deep analysis of their behavior under distinct training conditions would suffice for the intended scope of this project.
